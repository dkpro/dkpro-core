        public static final StanfordParser.DependenciesMode COLLAPSED
        Produce collapsed dependencies. This removes dependencies on specific function words (e.g. prepositions and conjunctions). The result not be a tree, e.g. it can include cycles and re-entrancies.
        Corresponding parser option: collapsed

      • COLLAPSED_WITH_EXTRA

        public static final StanfordParser.DependenciesMode COLLAPSED_WITH_EXTRA

        Corresponding parser option: not available

      • CC_PROPAGATED

        public static final StanfordParser.DependenciesMode CC_PROPAGATED
        Produce collapsed dependencies plus extra arcs for control relationships, etc. In this mode, depencendies are collapsed across coordination. This mode is supposed to produce the best syntactic and semantic representation of a sentence. The result may not be a tree (may contain cycles), but is a directed graph.
        Corresponding parser option: CCPropagated

      • CC_PROPAGATED_NO_EXTRA

        public static final StanfordParser.DependenciesMode CC_PROPAGATED_NO_EXTRA
        Produce dependencies collapsed across coordination. No extra dependencies for control relations are included.
        Corresponding parser option: not available

      • TREE

        public static final StanfordParser.DependenciesMode TREE
        Produce mostly collapsed dependencies that remain a tree structure. Several steps are omitted:
        1. no processing of relative clauses
        2. no xsubj relations
        3. no propagation of conjuncts
        Corresponding parser option: tree
