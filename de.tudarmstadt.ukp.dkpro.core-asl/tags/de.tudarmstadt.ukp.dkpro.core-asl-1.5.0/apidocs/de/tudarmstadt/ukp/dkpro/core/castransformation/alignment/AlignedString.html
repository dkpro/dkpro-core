      • inverseResolve

        public ImmutableInterval inverseResolve(ImmutableInterval i)
        For the given interval on the underlying data, get the corresponding interval on this level. Example: 11 11 111 11 012 345 6789 01 23 456 78 AD |111|22ZZ2|3333|44|55|YYY|55| UL |111|XX|22|ZZ|2|XXXXX|3333|XX|44|XXXX|5555|XXXX| 012 34 56 78 9 11111 1111 12 22 2222 2223 3333 01234 5678 90 12 3456 7890 1234 As you can see there is a YYY inserted in the AD. Otherwise some parts of the UL (marked "X") have been removed in the AD. Also an ZZ part has been added to UL Calling this method with start=22 end=30 should return [12, 18] as this is the interval 5 from UL plus the "Y" that has been inserted in AD. Generally: - if the start is within a deleted region, then find the next oblique segment in AD to the right and return its start position. - if the end is within a deleted region, then find the next oblique segment in AD to the left and return its end position. Anchors are always in UL. They are referenced from the ObliqueSegments in AD.
        Parameters:
        i - the interval on the underlying data.
        Returns:
        the corresponding interval in the view.
