Merges any Tokens that are covered by a given annotation type. E.g. this component can be used to create a single tokens from all tokens that constitute a multi-token named entity.

PARAM_LEMMA_MODE

public static final String PARAM_LEMMA_MODE
Configure what should happen to the lemma of the merged tokens. It is possible to JOIN the lemmata to a single lemma (space separated), to REMOVE the lemma or LEAVE the lemma of the first token as-is.

See Also:
Constant Field Values

PARAM_POS_VALUE

public static final String PARAM_POS_VALUE
Set a new POS value for the new merged token. This is the actual tagset value and is subject to tagset mapping. For example when merging tokens for named entities, the new POS value may be set to "NNP" (English/Penn Treebank Tagset).

See Also:
Constant Field Values

PARAM_POS_TYPE

public static final String PARAM_POS_TYPE
Set a new POS tag for the new merged token. This is the mapped type. If this is specified, tagset mapping will not be performed. This parameter has no effect unless PARAM_POS_VALUE is also set.
