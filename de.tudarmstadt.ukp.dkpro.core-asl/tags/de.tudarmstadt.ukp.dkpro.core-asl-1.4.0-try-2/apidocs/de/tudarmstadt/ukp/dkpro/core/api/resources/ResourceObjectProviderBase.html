public abstract class ResourceObjectProviderBase<M>
extends Object
implements HasResourceMetadata

Base class for resource providers that produce a resource from some URL depending on changing parameters such as language.

A component using such a provider sets defaults and overrides. The defaults should be set to sensible values with which the component should be able to work out of the box. For example the LOCATION may be set to "classpath:/resources/${language}/model.ser.gz".

The overrides should only be set if the user explicitly wants to override some settings.

Finally, parameters that may change, e.g. depending on the CAS content should be returned from getProperties().

The LOCATION may contain variables referring to any of the other settings, e.g. "${language}".

It is possible a different default variant needs to be used depending on the language. This can be configured by placing a properties file in the classpath and setting its location using setDefaultVariantsLocation(String) or by using setDefaultVariants(Properties). The key in the properties is the language and the value is used a default variant.
