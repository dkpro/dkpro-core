public class DictionaryAnnotator
extends org.uimafit.component.JCasAnnotator_ImplBase

Takes a plain text file with phrases as input and annotates the phrases in the CAS file. The annotation type defaults to NGram, but can be changed. The format of the phrase file is one phrase per line, tokens are separated by space: 

 this is a phrase
 another phrase

Johannes Hoffart
