getUrlAsFile

public static File getUrlAsFile(URL aUrl,
                                boolean aCache)
                         throws IOException
Make the given URL available as a file. A temporary file is created and deleted upon a regular shutdown of the JVM. If the parameter aCache is true, the temporary file is remembered in a cache and if a file is requested for the same URL at a later time, the same file is returned again. If the previously created file has been deleted meanwhile, it is recreated from the URL.

Parameters:
aUrl - the URL.
aCache - use the cache or not.
Returns:
a file created from the given URL.
Throws:
IOException - if the URL cannot be accessed to (re)create the file.

resolveLocation

public static URL resolveLocation(String aLocation,
                                  Object aCaller,
                                  org.apache.uima.UimaContext aContext)
                           throws IOException
Resolve a location (which can be many things) to an URL. If the location starts with classpath: the location is interpreted as a classpath location. Otherwise it is tried as a URL, file and at last UIMA resource. If the location is treated as a classpath or file location, an URL is only returned if the target exists. If it is an URL, it is possible that the target may not actually exist.

Parameters:
aLocation - a location (classpath, URL, file or UIMA resource location).
aCaller - the instance calling this method (for classpath loading).
aContext - a UIMA context.
Returns:
the resolved URL.
Throws:
IOException - if the target could not be found.
