Class FrequencyDistribution<T>

  • java.lang.Object
    • de.tudarmstadt.ukp.dkpro.core.api.frequency.util.FrequencyDistribution<T>
  • Type Parameters:
    T - the type of the samples
    All Implemented Interfaces:
    Serializable

    public class FrequencyDistribution<T>
extends Object
implements Serializable
    It is basically a mapping from samples (keys) to long values (counts).

    Suppose we want to record the number of occurrences of each word in a sentence, then this class can be used as follows: 

     FrequencyDistribution fd = new FrequencyDistribution();
 for (String word : "foo bar baz foo".split(" ")) {
     fd.inc(word);
 }
 System.out.println(fd.getCount("foo"));
    The last call to getCount(T) will yield 2, because the word "foo" has appeared twice in the given sequence of words.

    This class was inspired by NLTK's FreqDist.
